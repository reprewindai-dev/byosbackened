# Veklom Acquisition Surface

This directory is the static acquisition/strategic surface for `veklom.dev`.

The live marketplace/demo surface remains `backend/landing` and deploys to
`veklom.com`.

Deploy target:

- Cloudflare Pages project: `veklom-dev-acquisition`
- Custom domain: `veklom.dev`
- Redirect: `www.veklom.dev` -> `veklom.dev`
