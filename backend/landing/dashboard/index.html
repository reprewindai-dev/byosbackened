Quickstart
  1. Generate an API key above.
  2. Use the API key in `X-API-Key` for `/v1/exec` requests.
  3. Use JWT auth in the dashboard to manage AI and wallet controls.
curl -X POST https://api.veklom.com/v1/exec \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"prompt":"summarize this audit log entry","use_memory":false}'
